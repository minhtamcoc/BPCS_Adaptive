# bpcs_adaptive

Native Labtainer lab for Adaptive BPCS and checkerboard conjugation.

The lab uses original Labtainer checkwork/stoplab behavior. Do not install
custom wrappers.

Student-facing files:

```text
video.mp4
adaptive_encode.py
adaptive_decode.py
```

Expected grading:

```text
cw1: threshold_report.txt contains final_threshold.
cw2: message_blocks.json records conjugation.
cw3: adaptive_position.json records embedding metadata.
cw4: .bash_history records extraction to adaptive_extract_frames.
cw5: recovered_adaptive_secret.txt contains Adaptive BPCS conjugation.
```
